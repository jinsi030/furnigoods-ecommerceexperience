# Furnigoods — E-Commerce User Experience

## Overview

Furnigoods is a conceptual e-commerce experience designed to provide users with a seamless and intuitive furniture shopping journey.

The project focuses on user flows, product discoverability, usability, and functional prototyping to improve the overall shopping experience.

---

## Problem Statement

E-commerce platforms often overwhelm users with excessive information, poor navigation, and complicated checkout experiences.

The challenge was to design a streamlined shopping experience that improves product discovery and reduces friction during purchase.

---

## Objectives

- Improve product discoverability.
- Simplify navigation.
- Create a seamless checkout experience.
- Enhance overall usability.
- Design intuitive shopping flows.

---

## Design Process

### 1. User Research

Analyzed user expectations regarding:

- Product search
- Product comparison
- Checkout process
- Product categorization

### 2. Wireframing

Created wireframes to define:

- Navigation patterns
- Shopping flow
- Screen hierarchy

### 3. Interface Design

Designed user-friendly interfaces focusing on:

- Simplicity
- Clarity
- Consistency

### 4. Prototyping

Built interactive prototypes to validate user journeys.
